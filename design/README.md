# design/

UI/UX design review skills for visual quality, accessibility, and user experience. Applicable to any frontend framework, with additional MudBlazor-specific coverage.

## Skills

| Skill | Scope | Key checks |
|---|---|---|
| `design:ui` | UI implementation | Component consistency, WCAG AA contrast, semantic HTML, responsiveness |
| `design:ux` | UX flows | User flow completeness, feedback states, microcopy, cognitive load |
| `design:theme` | Theming | Dark/light token coverage, `prefers-color-scheme`, flash prevention |
| `design:landing` | Landing pages | Above-the-fold CTA, conversion flow, trust signals, mobile layout |
| `design:motion` | Animations | Purpose-driven motion, easing curves, GPU compositing, `prefers-reduced-motion` |
| `design:mudblazor` | MudBlazor | Native component usage, `MudThemeProvider`, CSS isolation, `MudGrid` breakpoints |
