# ops/

DevOps, infrastructure, and observability skills. Covers container configuration, CI/CD pipelines, IaC review, and monitoring design.

## Skills

| Skill | Scope | Key checks |
|---|---|---|
| `ops:docker` | Dockerfile / Compose | Multi-stage builds, layer caching, non-root user, `.dockerignore` |
| `ops:pipeline` | CI/CD (Actions, ADO) | Stage ordering, secret management, caching, parallelization |
| `ops:infra` | Terraform / Bicep / ARM | Resource naming, sensitive vars, state management, cost sizing |
| `ops:health-check` | ASP.NET Core | Liveness vs readiness, dependency checks, Kubernetes probes |
| `ops:alert` | Monitoring | Symptom-based alerts, severity tiers (P0/P1/P2), runbook stubs |
| `ops:dashboard` | Grafana / Azure Monitor | Panel layout, variables, thresholds, deployment annotations |
