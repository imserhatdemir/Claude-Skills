# ai/

AI development skills for prompt engineering, RAG pipeline review, and cost estimation. Model-agnostic but includes Claude-specific optimizations (prompt caching, batch API).

## Skills

| Skill | Scope | Key checks |
|---|---|---|
| `ai:prompt` | System prompts | Clarity, structure, safety guardrails, injection surface, token waste |
| `ai:rag` | RAG pipelines | Chunk sizing, embedding strategy, hybrid search, re-ranking, grounding |
| `ai:cost` | API cost planning | Token estimation, model comparison table, caching and batching opportunities |
